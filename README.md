# Inventory Warehouse System

Inventory Warehouse System adalah aplikasi berbasis web yang dibuat untuk membantu pengelolaan data barang di gudang, mulai dari pencatatan item, supplier, lokasi penyimpanan, hingga pergerakan stok masuk dan keluar.

Aplikasi ini dikembangkan sebagai bagian dari  project semester 5 menggunakan framework Laravel dan database MySQL.

---

## Fitur Utama
- Manajemen data barang (Item)
- Manajemen supplier
- Pengelolaan lokasi penyimpanan gudang
- Pencatatan stok barang per lokasi
- Purchase Order (PO)
- Sales Order (SO)
- Riwayat pergerakan stok (Stock Movement)
- Autentikasi pengguna (Admin / Staff)

---

## Teknologi yang Digunakan
- **Framework** : Laravel
- **Bahasa** : PHP
- **Database** : MySQL
- **Frontend** : Blade Template, HTML, CSS, Bootstrap
- **Version Control** : Git & GitHub
